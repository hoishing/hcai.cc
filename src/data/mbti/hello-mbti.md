## 什麼是 MBTI？

MBTI（Myers-Briggs Type Indicator）是一種廣泛使用的人格分類工具，將人格分為十六種類型。它基於四個維度：

- **E / I**：外向 vs 內向
- **S / N**：感覺 vs 直覺
- **T / F**：思考 vs 情感
- **J / P**：判斷 vs 感知

## 如何運用 MBTI？

MBTI 可以幫助你了解自己的溝通風格、工作偏好以及人際互動模式。不過，請記住它只是一個參考工具，而非絕對的標籤。
